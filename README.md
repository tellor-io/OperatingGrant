# Operating Grant

## Overview <a name="overview"> </a>  

This contract manages the vesting schedule for Tellor Inc. given a grant from the Tellor DAO. 

## Setting up and testing

Install Dependencies
```
npm i
```
Compile Smart Contracts
```
npx hardhat compile
```

Test Locally
```
npx hardhat test
```
